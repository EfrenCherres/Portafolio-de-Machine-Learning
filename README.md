# Portafolio-de-Machine-Learning
Portafolio de Machine Learning enfocado exclusivamente en Análisis Predictivo. Desarrollo de modelos avanzados para la detección de tendencias y pronósticos de mercado. Aplicación de algoritmos de aprendizaje supervisado y no supervisado para transformar datos históricos en proyecciones estratégicas de negocio. Por: Efren Cherres Bowen.
